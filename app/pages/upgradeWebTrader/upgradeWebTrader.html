Upgrade WebTrader

Upgrade Process

1. Navigate to the directory that contains the WebTrader application files.



2. Copy the folder the contains the WebTrader application files.



3. Paste the copied folder in a safe location for backing files and folders up. In this example we will paste the backup folder in the same directory as our live WebTrader application files. Rename the backup folder to reflect the contents.



4. Extract the attached WebTrader Upgrade build into the location of your choice.





5. Navigate to the directory that contains the newly extracted WebTrader Upgrade application files. Copy all files in this directory.



6. Navigate back to the directory that contains the current version of WebTrader (the same folder we copied in step two). Paste the files that you just copied into the directory that contains the current version of WebTrader.



7. If you are prompted about Replacing, Skipping, or Renaming files, select Replace existing files in the destination. This will ensure that your copy of WebTrader will contain all updated application files.



That’s it! Your WebTrader application has been upgraded to the latest build. If you are having issues, please review the Notes and Troubleshooting information below. If your issue still persists, contact us!

Notes

Note: If the entire WebTrader application folder was deleted before uploading/copying the new build, the config.js file will need to be uploaded/copied. This file can be grabbed from the original WebTrader application folder that was backed up in step 2. The path to access this file is: /js/config.js.

Note: The chart license is already included in the attached upgrade build; no additional configuration is necessary for this component.

Troubleshooting

I still see the old WebTrader!

Clear the web browser cache on your machine.

I do not see any charts, but everything else works.

Your domain name is not allowed by the Chart License. Please send us the full domain name so that we are able to send you an updated build.

The WebTrader doesn’t load at all!

Open the developer console and refresh the WebTrader page. Gather the error messages from the console and send them to us so that we are able to further investigate the issue.
