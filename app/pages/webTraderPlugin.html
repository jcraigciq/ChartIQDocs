WebTrader MT4 Plugin Installation Guide

Instructions

1. Download the plugin.

2. Unzip the compressed file.

3. Copy the qlwebtrader-2k8s.dll file from the unzipped folder.

4. Place the qlwebtrader-2k8s.dll file in the following directory: c:/MetaTrader4Server/plugins/

5. Restart the MT4 server.

6. Login to MT4 Administrator and configure the parameters shown in the next section.

MT4 Administrator Parameters

WT Balance operations
n - flag to allow balance operations from WebTrader
(0 - denied, 1 - permitted)

Allowed off-quote time
n - allowed off-quote time in seconds

IP Whitelist
127.0.0.1 - comma separated list of IPs from which connection to the plugin permitted

Max command queue size
n - the size of the queue for the commands requested from the WebTrader Web Engine
default is set to 500, increase in the case of high requests flow (>1k per second)

Max connections
n - max number of incoming connections from the WebTrader Web Engine.
(at a minimum the value should be the number of Web Engine nodes times 2)

Max price diff (%)
n - max difference between the order and market price in %

Number of workers
n - number of workers to handle requests
(at a minimum the value should be 2)

Port to Listen
n - the port which plugin listens for incoming connections from the WebTrader Web Engine

Price IE from market
0 - use the price from WebTrader for Instant Execution order
1 - for Instant Execution order takes the price from the market

WT Reporting
n - reporting level for WebTrader
(0 - reporting max, 5 - reporting min)

Web API key
some_string - used for WebTrader Web Engine authorization
(leave blank if whitelist is set)

If ChartIQ is hosting your WebTrader Middleware...

7. After the plugin parameters are configured, copy the plugin.ini file from the c:/MetaTrader4Server/plugins/ directory.

8. Please send the plugin.ini file back to ChartIQ to finalize connecting your MT4 server to the hosted WebTrader Middleware.
