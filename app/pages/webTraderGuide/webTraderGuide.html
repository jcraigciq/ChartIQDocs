WebTrader Guide

Introduction

WebTrader is an HTML5 application developed in Angular 1.x. Each user interface (UI) component is represented by an Angular "partial" (a self-contained block of HTML code) and a corresponding Angular controller (a js file that contains business logic and binding code). Angular data-binding automatically updates the state of the UI.

A set of Angular "services" provide connectivity and utility functions. The connectivity services use the socket.io library to communicate with ChartIQ's Middleware for MT4.

Market Watch

Description: The Market Watch component displays real time quotes for all Symbols that the user has access to trade. This component also includes features to start a New Order/New Pending Order dialog or plot the Symbol in the ChartDock. 1. The trading portion of this component has two different behaviors depending on if the user has One-Click Trading enabled or not. If One-Click Trading is enabled then the user is able to input the Volume, SL PIPS, TP PIPS and then click Buy/Sell to instantly trigger the transaction. If One-Click Trading is disabled, then clicking Buy/Sell will trigger the New Order/New Pending Order dialog. 2. The user can also trigger the New Order/New Pending Order dialog by pressing the icon in the top right hand corner of the expanded Symbol dialog. 3. The icon enables you to plot the Symbol on the ChartDock. After clicking the icon, a context menu will appear that prompts you for the chart interval to apply to the x-axis of the chart. 4. The different chart interval options are:  1 Minute  5 Minutes  15 Minutes  30 Minutes  1 Hour  4 Hours  1 Day  1 Week  1 Month

ChartDock

Description: The ChartDock enables traders to keep multiple charts (up to 10) running in the same browser window. The flexible interface allows users to slide and stack charts, or let the application automatically dock charts in a grid pattern. Charts on the ChartDock can be quickly manipulated on the fly by right-clicking them to access an additional options menu. 1. -Displays the Symbol and chart interval in the top left hand corner of each chart. 2. - Displays the Spread for the current Symbol in the top right corner of the chart. 3. - Enables the user to zoom the individual charts in the ChartDock in and out. 4. - Opens the Chart Settings dialog, allowing the user to edit individual charts in the ChartDock. Configurable options include Style, Type, and Studies. a. b. Style options:  Candle  Bar  Line  Mountain  Colored Bar  Hollow  Baseline Delta  Vol Candle c. Type options:  Heikin-Ashi  Kagi  Line Break  Point & Figure  Range Bars  Renko  None d. Studies options:  Refer to the Chart Studies dialog in WebTrader to see all Studies available. 5. - Toggles the trading from the chart dialog.  *NOTE: One-Click Trading has to be enabled in order to utilize trading from the chart. 6. - Right click a chart in the ChartDock to access a chart context menu. a. - Triggers a New Pending Order dialog for the Symbol the user is viewing. b. - Triggers a New Order dialog for the Symbol the user is viewing. c. - Triggers the Chart Studies dialog. d. - Opens another context menu to change the chart interval.

Statistics

Description: The Statistics component displays various metrics that describe the user’s trading performance. This information is accessible by clicking the icon in the top right of the application.

Footer

Description: The Footer is split into two sections. The left section shows the Open, Close, High, Low, and Volume based on the cursors position on any chart in the ChartDock. The right section shows the Account Number, Leverage available to the account, Credit available to the account, and the name of the Server that the user is connected to.

Dialogs

Login

Description: Standard Login screen with Server, Account, and Password inputs. Description: Standard Login screen with Email and Password inputs.

New Order

Description: This dialog enables the user to place a New Order. The user can choose to click New Pending to trigger the New Pending Order dialog. 1. - Enables the user to toggle between New Order and New Pending Order. 2. - Dropdown selector for picking the Symbol. 3. - Input for the Volume size of a trade. 4. - Enables the user to select a Side for the transaction, Buy or Sell. 5. - Input for setting the Stop Loss value. 6. - Shows current loss based on the Stop Loss value. 7. - Input for setting the Take Profit value. 8. - Shows current profit based on the Take Profit value. 9. - Comment for the trade (logged in MT4). 10. - Displays the Bid and Ask for the Symbol selected. 11. - Buy/Sell will display at the bottom right of the dialog depending on the Side. Click Buy/Sell to execute the trade.

New Pending Order

Description: This dialog enables the user to place a New Pending Order. The user can choose to click New Order to trigger the New Order dialog. 1. - Enables the user to toggle between New Order and New Pending Order. 2. - Dropdown selector for picking the Symbol. 3. - Input for the Volume size of a trade. 4. - Drop down menu for choosing the Type of transaction, these include: a. Buy Limit b. Sell Limit c. Buy Stop d. Sell Stop 5. - Input for user to set the Price condition. 6. - Input for setting the Stop Loss value. 7. - Shows current loss based on the Stop Loss value. 8. - Input for setting the Take Profit value. 9. - Shows current profit based on the Take Profit value. 10. - Input for user to set the date and time that the New Pending Order will expire on. 11. - Comment for the trade (logged in MT4). 12. - Displays the Bid and Ask for the Symbol selected. 13. - Click to Place the Pending Order.

Modify Pending Order

Description: A dialog menu for enabling the user to modify a pending order. 1. - Enables the user to toggle between New Order and New Pending Order. - Drop down menu for the user to select which Pending Order they wish to modify. 2. - Drop down menu for choosing the Type of transaction, these include: a. Buy Limit b. Sell Limit c. Buy Stop d. Sell Stop 3. - Input for user to set the Price condition. 4. - Input for setting the Stop Loss value. 5. - Shows current loss based on the Stop Loss value. 6. - Input for setting the Take Profit value. 7. - Shows current profit based on the Take Profit value. 8. - Input for user to set the date and time that the New Pending Order will expire on. 9. - Comment for the trade (logged in MT4). 10. - Displays the Bid and Ask for the Symbol selected. 11. - Updates the Pending Order with the latest inputs from the user. 12. - Deletes the Pending Order completely.

Modify Position

Description: A dialog menu for enabling the user to update or close an open position. 1. - Enables the user to switch to a New Order or New Pending Order. 2. - Drop down menu for the user to select which Position they wish to modify. 3. - Input for setting the Stop Loss value. 4. - Shows current loss based on the Stop Loss value. 5. - Input for setting the Take Profit value. 6. - Shows current profit based on the Take Profit value. 7. - Displays the Bid and Ask for the Symbol selected. 8. - Update the position with the latest inputs from the user. 9. - Close the open position.

Settings

Description: The Settings dialog enables the user to configure general settings of the WebTrader application. 1. Display Ask Line - Enables or Disables Ask Lines in the ChartDock. 2. Display Bid Line - Enables or Disables Bid Lines in the ChartDock. 3. One-Click Trading – Enables or Disables the One-Click Trading option. 4. Play Sounds – Enables or Disables all sounds related to WebTrader. 5. Trades On Chart – Enables or Disables user trades displaying in the ChartDock.

Chart Studies

Description: This is the Chart Studies dialog. Select an indicator to bring up additional settings and enable it to be plotted on a chart in the ChartDock.

One-Click Trading

Description: The One-Click Trading dialog allows the user to enable the functionality of One-Click Trading, which executes trades immediately after the order is placed. This feature should be utilized by advanced users only. This feature can be enabled from two different areas in the application: 1. - Inside of the Settings dialog. 2. - Toggle button in the top right hand corner of the application.
