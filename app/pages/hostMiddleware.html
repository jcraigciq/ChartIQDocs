Host Middleware for MT4

System Requirements

Our middleware is written in nodeJS/socket.io. This provides a Web socket API, allowing you to authenticate MT4 users, retrieve their data, subscribe to real time quotes, trade, and receive other updates from MT4. NginX is used as a reverse proxy and load balancer.

Determining the server specifications that you need to host the middleware depends on a few simple questions:

  1. How many MT4 servers are you connecting to?
  2. How many concurrent users will you have trading on each MT4?
  3. How many instruments will you be trading on each MT4?
Example:

Meeting the following system requirements will host middleware for 1 MT4 Server with 160 concurrent users and 100 subscribed symbols.

System Requirements
  • Operating System: Windows Server 2012
  • Processor (CPU): 2.0+ GHz processor (minimum of 4 cores)
  • Memory: 2 GB RAM
  • Storage: 10 GB hard drive
